# Agroecology AI Experiments

## À propos

Ce dépôt présente mes premières explorations autour de l'intelligence artificielle appliquée à l'agroécologie, aux savoirs locaux et aux territoires.

L'objectif est de comprendre comment les systèmes d'IA générative et les architectures RAG (Retrieval-Augmented Generation) peuvent contribuer à améliorer l'accès à l'information agricole tout en valorisant les connaissances locales.

## Objectifs

* Explorer les systèmes RAG
* Comprendre la recherche documentaire assistée par IA
* Expérimenter les embeddings et la recherche sémantique
* Réfléchir à l'utilisation des langues locales dans les systèmes d'IA
* Valoriser les connaissances agroécologiques

## Contenu du dépôt

### Agroecology_AI_RAG_Experiment.ipynb

Notebook présentant une expérimentation simple de recherche documentaire basée sur des embeddings.

Réflexion personnelle sur l'intelligence artificielle, l'agroécologie et les savoirs locaux.
